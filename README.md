# Scrawl-Manga
Scrawl Manga is get manga from some web manga, it will write to file pdf, used for another device when offline.

Using python 3.5.5

Syntax:
    FeedMangaToPdf.exe url
    FeedMangaToPdf.exe <file.txt>  // content many url
